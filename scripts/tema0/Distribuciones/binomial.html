Sea \(X = B(n = 30,\ p = 0.6)\)

Función de densidad

\[f(k) = {n \choose k} p^k (1-p)^{n-k} \]

Función de distribución

\[F(x) = \left\{\begin{array}{cl} 0 & \text{si } x \lt 0 \\ \sum_{k=0}^x f(k) & \text{si } 0 \le x \lt n \\ 1 & \text{si } x \ge n \end{array}\right.\]

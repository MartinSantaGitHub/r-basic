Distribuci√≥n Normal

Supongamos que \(X\sim N(3,0.5)\) entonces podemos estudiar sus par√°metros.

En R

mu = 3
sig = 0.5
x = seq(0, 6, by = 0.05)

hist(rnorm(100000, mean = mu, sd = sig), freq = FALSE, main = NULL, xlab = '', xlim = c(0,6))
lines(x, dnorm(x, mean = mu, sd = sig), type = 'l', col = 'red', lwd = 5)
legend("topright",
       col = c("red"),
       legend=c("Funci√≥n de Densidad\nde una N(3,0.5)"),
       lwd=5,
       bty="n"
       )

plot(pnorm(q = x, mean = mu, sd = sig), type = 'l', xaxt = 'n', lwd = 2, col = 'blue',
     main = "Funci√≥n de Distribuci√≥n Acumulada de una N(3,0.5)",
     xlab = '',
     ylab = '')
axis(1, at = c(0,20,40,60,80, 100, 120), labels = c('0','1','2','3','4','5','6'))

qnorm(0.25, mean = mu, sd = sig)

En Python

Sys.setenv(RETICULATE_PYTHON = "/Python38")
from scipy.stats import norm
import matplotlib.pyplot as plt
import numpy as np

fig, ax = plt.subplots(1,1)

mu = 3
sig = 0.5
rv = norm(loc = mu, scale = sig)
x = np.linspace(0, 6, 120)
r = rv.rvs(size = 100000)
mean, var, skew, kurt = rv.stats(moments = 'mvsk')

print(f"Media {mean}")
## Media 3.0
print(f"Varianza {var}")
## Varianza 0.25
print(f"Sesgo {skew}")
## Sesgo 0.0
print(f"Curtosis {kurt}")
## Curtosis 0.0
ax.plot(x, rv.pdf(x), 'r-', lw = 5, alpha = 0.6, label = "Funci√≥n de Densidad\nde una N(3,0.5)")
ax.hist(r, density = True, histtype = 'stepfilled', alpha = 0.2, bins = 25)
